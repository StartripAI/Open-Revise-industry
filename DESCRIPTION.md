# Repository Description

## GitHub About (Short)
Evidence-gated DOCX revision toolkit for high-stakes Q&A: source gate, tracked changes, and auditable Q-to-source mapping.

## GitHub About (中文)
一个“有证据才改”的 DOCX 问答修订工具：先过来源门禁，再做修订，并输出可审计的 Q-source 映射。

## One-line Value
Revise only when evidence is sufficient; otherwise explicitly mark unavailable evidence instead of guessing.
